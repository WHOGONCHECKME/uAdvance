Python 1A

Kickstart your journey into programming with Python, a language known for its simplicity and versatility.
Participants will learn the basics of Python including:

  • Introduction to Python and its ecosystem
  • Basic syntax, data types, and variables
  • Control structures: if statements, loops, and functions
  • Basic data structures: lists, dictionaries, sets, and tuples
  • Working with files and directories
  • Introduction to error handling and debugging

    • The amount for the Python 1A single day session is: $670 + GST

    Python 1B

    This next level session assumes familiarity with Python basics and focuses on practical applications, especially in data science:

  • Introduction to libraries like Pandas and NumPy
  • Data manipulation and cleaning techniques
  • Introduction to data visualization with Matplotlib and Seaborn
  • Writing and using custom functions and modules
  • Exploratory data analysis
  • Basic machine learning with scikit-learn

    • The amount for the Python 1B single day session is: $670 + GST
