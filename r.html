R 1A

For those that want to advance their toolkit with a more powerful powerful software. You will learn to manipulate data, visualise data and automate reports
Participants will learn and introduction to R including the following key concepts:

  • Introduction to R and RStudio
  • Basic syntax and data types
  • Data structures: vectors, lists, matrices, data frames and factors
  • Data import/export: reading and writing data
  • Data manipulation using dplyr
  • Data visualisation using ggplot2
  • Introduction to data governance and best practices

    • The amount for the R 1A single day session is: $670 + GST

    R 1B

    This is a more applied day, where techniques learned will be more applied to financial services with practical problems being tackled, basic concepts from 1A are assumed knowledge and we dive right in to:

  • Using R markdown for report writing
  • Building dashboards
  • Handling financial data
  • Basic statistical analysis

    • The amount for the R 1B single day session is: $670 + GST

    Course Materials

    This course is an adaptation of Harvard university’s CSS50’s Introduction to Programming with R.
    It has been adapted for the financial services industry in Australia.
    If students are looking to attain a verified certificate or transfer credit and accreditation, this course will supplement and prepare them for the course’s problem sets and final project.
