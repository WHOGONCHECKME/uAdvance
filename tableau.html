Tableau 1A

Learn the fundamentals of Tableau and start creating visual stories with your data.
The session covers:

  • Getting started with Tableau interface
  • Connecting to spreadsheets and databases
  • Building basic visualizations and dashboards
  • Filters, sorting and grouping
  • Using calculated fields
  • Sharing workbooks and publishing to Tableau Public
